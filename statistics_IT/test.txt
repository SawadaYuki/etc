#四分位, 標準偏差

describe() メソッドをで、件数 (count)、平均値 (mean)、標準偏差 (std)、最小値(min)、第一四分位数 (25%)、中央値 (50%)、第三四分位数 (75%)、最大値 (max) を確認することができます。

>>> import pandas as pd
>>> import numpy as np
 
# データフレーム df を作成・表示
>>> df = pd.DataFrame(np.random.randn(6,4), columns=list('ABCD'))
>>> df.describe()
